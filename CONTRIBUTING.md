# Contributing to pulumi-go-provider

## Release Process

To cut and release a new version of this framework:

1. From the [Releases page](https://github.com/pulumi/pulumi-go-provider/releases), click **Draft a new release**.
2. Create a new tag with a [semantic version](https://semver.org/) (e.g., `v1.2.3`) targeting `main`.
3. Generate release notes and publish.
